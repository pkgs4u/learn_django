## Setup Project

<p>Clone the project with the url "https://github.com/pkgs4u/learn_django.git".</p>

### Virtual Environment
<p>Configure virtual environment if not.</p>

```bash
$ python -m venv venv
$ venv\Scripts\activate.bat
```

### Install Django

```bash
$ pip install django
```
